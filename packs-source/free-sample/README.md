Copy the `agents/` folder to your Clawdbot workspace.

### 2. Update clawdbot.json
Add the following to your `agents` array in `clawdbot.json`:

```json
{
  "agents": [
    {
      "id": "helper",
      "name": "Helper",

      "channels": ["general"]
    }
  ]
}
```

### 3. Restart Gateway
```bash
clawdbot gateway restart
```
